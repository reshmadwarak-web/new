# Aether - Premium Creative Developer Portfolio

A sleek, modern, high-performance single-page developer portfolio website. Built with custom vanilla CSS, semantic HTML5, and interactive vanilla JavaScript. Featuring a futuristic cybernetic aesthetic with glassmorphism panels, glow variables, and interactive mockups.

![Design Showcase](https://images.unsplash.com/photo-1507238691740-187a5b1d37b8?auto=format&fit=crop&w=1200&q=80)

## Features

- 🌌 **Futuristic Cybernetic Theme**: Premium dark mode design utilizing deep slate blues, dark obsidian bases, and bright neon gradients (cyan and purple).
- 🧩 **Interactive Terminal Emulator**: A styled command-line interface in the hero section displaying real-time command logs on page load.
- 🎨 **Interactive UI Mockups**: Projects feature live interactive web graphics (such as animated nodes for machine learning or floating e-commerce swap modules) rather than flat screenshots.
- 💫 **Mouse Follow Glow**: A smooth radial gradient follower that hovers behind elements on desktop browsers.
- 📜 **Scroll Spy Navigation**: Navbar active link states automatically update as you scroll through different sections of the page.
- ⚡ **Performance & Scalability**: Pure HTML, CSS, and JS with zero framework dependencies. Blazing fast load times and completely responsive across mobile, tablet, and ultra-wide screens.
- 💼 **Professional Roadmap**: Vertical timeline representing roles, tasks, and tech stack tags.
- ✉️ **Functional Contact Form Simulator**: Standard form validation with custom loading feedback states.

## Getting Started

### Run Locally

Since this is a vanilla HTML/CSS/JS application, there is no build step.

1. Clone or download the repository.
2. Open `index.html` directly in any web browser.
3. Alternatively, run a local development server for the best experience:
   ```bash
   # Using Python (built-in)
   python -m http-server 8000
   
   # Using Node (npx)
   npx http-server
   ```
4. Access the site at `http://localhost:8000` or the port provided by the tool.

## Deployment to GitHub Pages

You can host this website completely for free using **GitHub Pages**:

1. Create a new public repository on [GitHub](https://github.com).
2. Initialize git in your local project folder and push it:
   ```bash
   git init
   git add .
   git commit -m "Initial commit of premium developer portfolio"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
   git push -u origin main
   ```
3. On GitHub, navigate to your repository **Settings**.
4. Scroll down to the **Pages** tab on the left-hand menu.
5. Under **Build and deployment**, set the Source to **Deploy from a branch**.
6. Select the `main` branch and `/ (root)` folder, then click **Save**.
7. In a minute, your portfolio will be live at `https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/`!
